# 🌐 My Portfolio

A modern, responsive, and interactive personal portfolio website developed to showcase my skills, projects, education, certifications, and professional journey as a Frontend Developer. This portfolio serves as my digital resume, allowing recruiters and visitors to explore my technical expertise, completed projects, and contact information in one place.

🔗 **Live Website:** https://my-portfolio-9i02.onrender.com

📂 **GitHub Repository:**  https://github.com/rajanayak8336/My-Portfolio.git

# 📖 Project Description

This portfolio website is designed to represent my professional profile as a Frontend Developer. It provides a clean and attractive user interface where visitors can learn about me, explore my technical skills, view my featured projects, download my resume, and contact me directly.

The website is fully responsive, ensuring a seamless experience across desktops, tablets, and mobile devices. It demonstrates my ability to build modern, user-friendly web applications using current frontend technologies.
